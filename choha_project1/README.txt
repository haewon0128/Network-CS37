1. First type: make all 

2. Type: python chatserv.py [port number]
   the port number is the port that you want.

3. On another window, type: chatclient [serverhost] [serverport]
   the server host should be the server that server side run on such as flip1, flip2, flip3
   the server port should be same with the server run on. 

4. The client will get handle from user than send first auto message to server

5. whenever you want to quit, type: \quit
